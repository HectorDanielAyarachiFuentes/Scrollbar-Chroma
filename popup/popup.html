🔓 Desbloquear Funciones Completas

Firefox limita algunas funciones por defecto. Seguí estos pasos para habilitar redondeado, texto y grosores exactos en todas las barras.

1️⃣

Abrí About Config

Escribí about:config en la barra de direcciones y presioná Enter.

2️⃣

Buscá la opción

Buscá layout.css.webkit-scrollbar.enabled y activalo en true.

3️⃣

Recargá las páginas

Recargá todas las pestañas abiertas (F5) para aplicar los cambios.

✅ Solo necesitás hacerlo una vez. Firefox recordará la configuración entre sesiones.
